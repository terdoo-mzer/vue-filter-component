# vue-project

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### To run local json server
install globally by running  ```npm install -g json-server```
### Start the server
```json-server --watch db.json --port 5000```
